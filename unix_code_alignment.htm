ACM project unix code

Alignment of RNAseq profiles to the human transcriptome

Login in the cluster

> ssh -p 12034 acmo02@cluster.di.fct.unl.pt

Create a folder

> mkdir Beatriz

> mkdir Beatriz/project

> cd project

Copie necessary files to new directory

The human transcriptome (GENCODE v39)

> cp -r /mnt/share/acmo/AppCOmics_data/Genomes .

Reserve a resource in interactive mode

> tmux

> oarsub -l walltime=3:00 -I

Align the RNAseq data to the human transcriptome

The alignment for all the samples was done with the bash code on the file multiAlignment.sh and as input used a file with the name of the samples and the fasta file name

Make the code executable

> chmod +x multiAlignment.sh

Run

> ./multiAlignment.sh targets.txt

A folder for each sample was created with the output abundance.h5 abundance.tsv run_info.json

Check the output

> head Arid1bKD_1/abundance.tsv
